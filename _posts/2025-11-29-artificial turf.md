不同材質都有各自的**防火等級（Flammability Rating）**標準，最常見的是 **UL 94**（美國 Underwriters Laboratories 制定），也是塑膠零件最常用的防火規範。以下是整理給你快速查閱用的：

---

# 🔥 **UL 94 防火等級（塑膠常用）**

UL 94 主要分為兩類：

## **1. 垂直燃燒（Vertical – V 等級）**

**最常用、最被要求的防火規格**

| 等級      | 阻燃能力    | 說明                          |
| ------- | ------- | --------------------------- |
| **V-0** | ⭐⭐⭐（最高） | 點火後 **10 秒內熄滅**，不得有滴落物引燃棉花  |
| **V-1** | ⭐⭐      | 點火後 **30 秒內熄滅**，不可有可燃滴落物    |
| **V-2** | ⭐       | 點火後 **30 秒內熄滅**，但允許有滴落物引燃棉花 |

🔍 應用：電子零件外殼、家電、儀器內部塑膠件。

---

## **2. 水平燃燒（Horizontal – HB 等級）**

| 等級     | 說明               |
| ------ | ---------------- |
| **HB** | 最低等級，水平燃燒速度需符合規範 |

🔍 應用：一般低危險性的外殼、裝飾部件。

---

# 🧪 **其他 UL 94 特殊等級**

| 等級            | 特性           | 說明                     |
| ------------- | ------------ | ---------------------- |
| **5VA**       | 比 V-0 更嚴格    | 火焰拿開後 60 秒內熄滅、**不可穿洞** |
| **5VB**       | 介於 5VA 與 V-0 | 可穿洞但仍需迅速熄滅             |
| **VTM-0/1/2** | 用於薄膜（薄材）     | 對薄膜類材料的阻燃測試            |

---

# 🧱 **常見工程塑膠的防火能力（一般趨勢）**

> ⚠️「材質本身」與「配方、添加阻燃劑」影響很大，所以以下是 **典型、不加阻燃劑** 的情況。

| 材質                             | 典型防火表現                 | 阻燃版本          |
| ------------------------------ | ---------------------- | ------------- |
| **ABS**                        | 通常是 HB                 | 可做到 V-0（加阻燃劑） |
| **PC（聚碳酸酯）**                   | 天生阻燃，通常是 V-2 或 V-0     | 容易做到 V-0      |
| **PC+ABS**                     | V-0 容易                 | 多用於 3C 外殼     |
| **PA（尼龍）**                     | 常有滴落，基本 UL 94 HB 或 V-2 | 加玻纖/阻燃劑可到 V-0 |
| **PBT**                        | 容易滴落                   | 加阻燃劑可到 V-0    |
| **BMC（Bulk Molding Compound）** | 普遍有很好的阻燃，常見 V-0        | 多用電器外殼        |
| **ASA / PP / PE**              | 一般 HB，燃燒性高             | 加阻燃劑後才能好很多    |

---

人工草皮（Artificial Turf）的**防火等級**不像塑膠零件有 UL94，那是電子零件用的。
人工草皮的防火評定主要依照 **建築、地板、公共安全** 等規範，不同國家使用不同標準。

下面整理出 **最常引用、你最可能遇到的防火等級**。

---

# 🔥 人工草皮主要防火標準（依國際/台灣）

## **1. ASTM（美國）**

### 🔸 **ASTM D2859 – 住宅地毯防灼測試（Pill Test）**

**人工草皮最常用的防火標準。**
要求是：

* 火源（藥片）點燃後，燃燒圈不得超過 **1 inch（約 2.5 cm）**
* 通過即可視為 **具阻燃性**

人工草皮廠商常標示為：
**Pass ASTM D2859（阻燃）**

---

## **2. NFPA（美國消防協會）**

### 🔸 **NFPA 701 – 紡織布料阻燃測試**

人工草皮屬地板材，不一定使用此標準，但有些廠商仍提供。

---

## **3. EN（歐盟標準）**

### 🔸 **EN 13501-1 – 建築材料防火等級**

人工草皮常按此歐洲建築標準測試。

等級從高到低：
**A1 → A2 → B → C → D → E → F**

人工草皮常見等級：

* **Cfl-s1**（較佳）
* **Dfl-s1**（中等）
* “fl” 表示 **flooring（地板材料）**
* **s1** 代表煙量低（最好）

例如：
**EN13501-1 Cfl-s1** = 地板材料，低煙量，較高阻燃。

---

## **4. AS 1530.3（澳洲）**

評估：

* 火焰擴散（FSI）
* 烟量（SMOG）
* 熱釋放

人工草皮常標：
**AS 1530.3 Pass**

---

# 📌 人工草皮常見的防火等級 “實際情況”

典型人工草皮（PE / PP）本身 **可燃**，因此需要添加 **阻燃添加劑（FR）** 才能通過以下測試：

* **ASTM D2859（最基本）**
* **EN 13501-1 Cfl-s1 / Dfl-s1（較高要求）**

**真正高防火**的人工草皮通常用於：

* 展示會場
* 商場地板
* 大型公共空間
* 逃生路線周邊

一般居家草皮不一定有阻燃等級。

## 🧱 耐燃／不燃等級 — 材料分級

根據 CNS 14705 + 建築技術規則，常見材料防火等級如下：

| 等級             | 名稱   | 特性 / 適用範圍                                                                                 |
| -------------- | ---- | ----------------------------------------------------------------------------------------- |
| **耐燃一級**（不燃材料） | 不燃   | 幾乎不燃燒、不會釋放有害氣體／煙霧。常用於防火牆、防火構造、避難梯間、安全出口、消防通道等關鍵部位。常見材質：混凝土、磚、鋼鐵、石材、玻璃、砂漿等。 ([Chunlyn][2]) |
| **耐燃二級**（耐火板）  | 中等耐燃 | 可在一定時間內延緩燃燒／火焰蔓延。常用於一般公共空間內裝、隔間牆、天花板等。材質例如木絲水泥板、耐燃石膏板等。 ([Chunlyn][2])                    |
| **耐燃三級**（耐燃材料） | 基本耐燃 | 防火能力較弱，但比普通可燃材好，適合用於火災風險低、非結構性、非避難路線的裝修材或附著物。材質可能包含某些耐燃合板、耐燃塑膠板等。 ([Chunlyn][2])          |

**實務應用範例**：

* 逃生通道、樓梯間、避難梯間、安全出口、排煙通道 → 必須使用「耐燃一級」不燃材料。 ([Chunlyn][2])
* 一般室內裝潢、隔間牆、天花板 → 可考慮使用「耐燃二級」或「耐燃三級」，但若是高樓、公共場合、地下室可能有更嚴格要求。 ([Chunlyn][2])

# 自行測試某種材料的防火性能

---

# 🔥 1. 基本的「點火反應」測試（最重要）

## **目的：看材料是不是「一點就燃」、「自燃」、「滴落」**

### 適用：人工草皮、遮陽布、塑膠片、PVC、EVA、PE、PP

### **做法：**

1. 剪下**5×10cm** 材料一片。
2. 用 **打火機火焰** 接觸邊緣 **2 秒後移開**。
3. 觀察：

### **判讀：**

| 現象           | 結果判定                 |
| ------------ | -------------------- |
| 一點就爆燃、快速蔓延整片 | **極度可燃（危險）**，不可用於雨棚  |
| 火源拿開後馬上熄滅    | **具阻燃特性**            |
| 慢慢燃燒，但不蔓延    | **中等阻燃**             |
| 邊燒邊滴熔融塑膠滴下   | **高度危險**（高層外牆火災都因熔滴） |
| 無黑煙、熄滅快      | 相對安全                 |

人工草皮大多是 PE / PP → **通常會快速燃燒並熔滴**。

---

# 🔥 2. 垂直燃燒測試（簡易版 ＝ 類似 UL94）

### 方法：

1. 把材料垂直夾住（用夾子）。
2. 火焰接觸底部 10 秒，然後移開。
3. 測量 **離火後持續燃燒秒數**、是否到頂、是否掉熔滴。

### 判讀：

| 行為          | 對應等級（概略）         |
| ----------- | ---------------- |
| 離火立即熄滅      | 類似 V-0（非常好）      |
| 燒 10 秒以內熄滅  | 類似 V-1           |
| 燒 30 秒以上、滴落 | **V-2 以下（非常可燃）** |

人工草皮通常落在 **最差** 的等級（可燃塑膠）。

---

# 🔥 3. 水平火焰蔓延測試（簡易版 ≈ flooring flammability）

### 作法：

1. 橫放材料。
2. 點燃其中一角。
3. 測量火焰蔓延距離與時間。

### 判斷：

* **10 秒內蔓延超過 2 cm → 危險**
* **能自行熄滅 → 有阻燃劑**
* **蔓延速度極快 → 完全不可用在外部雨棚**

---

# 🔥 4. 熱源測試（模擬太陽熱）

人工草皮放室外，表面溫度夏天可達 **60–75°C**。

測法：

1. 用熱風槍調 120°C、距離 10 cm 烘 30 秒。
2. 看是否：

   * 變形
   * 軟化
   * 冒煙
   * 溶解

如果 **在 120°C 就冒煙或軟化** → 屬高危材料。

---

# 🔥 5. 煙量判定（簡易版）

引燃材料後觀察：

* **白煙、多且刺激眼睛** → 塑膠完全燃燒
* **濃黑煙** → 火災時非常危險（阻擋視線、窒息）
* **很少煙** → 表現佳

人工草皮燃燒 → **濃黑煙**，非常危險。

---

# 🔥 6. 氣味判定

燃燒後的味道可大概判斷材料：

| 氣味    | 推論              |
| ----- | --------------- |
| 藥味、酸味 | PVC（毒氣多）        |
| 蠟油味   | PE / PP（人工草皮常見） |
| 嗆味、刺鼻 | 劣質背膠或膠水         |

---

# 📝 7. 多數雨棚材料其實是「可燃」的

常見雨棚覆蓋物：

| 材料       | 是否可燃？     |
| -------- | --------- |
| PE 人工草皮  | ❌ 非常可燃    |
| PP 人工草皮  | ❌ 非常可燃    |
| PVC 布    | ⚠️ 可燃、有毒煙 |
| PC（中空板）  | ⚠️ 可燃     |
| 壓克力板     | ⚠️ 可燃     |
| 鋁複合板     | ⚠️ 塑膠心材可燃 |
| 金屬板（鋁、鐵） | ⭕ 不燃      |
| 玻璃       | ⭕ 不燃      |

所以如果你測試結果很差 → **是正常的**。

---

# 🔥 最後結論（最重要給你）

雨棚上若要鋪人工草皮這種「可燃塑膠」，無論怎麼測幾乎都會呈現：

* **快速燃燒**
* **熔滴**
* **濃黑煙**

→ **不建議使用在雨棚（外部建築）上**。

即使你買到阻燃草皮，它的背膠和纖維仍然屬 PE / PP，只是延緩燃燒而已，不可能達到「不燃」等級。

A 级：不燃性建筑材料，几乎不发生燃烧的材料。

A1级：不燃，不起明火

A2级：不燃，要测量烟，要合格。

B1级：难燃性建筑材料：难燃类材料有较好的阻燃作用。其在空气中遇明火或在高温作用下难起火，不易很快发生蔓延，且当火源移开后燃烧立即停止。

B2级：可燃性建筑材料：可燃类材料有一定的阻燃作用。在空气中遇明火或在高温作用下会立即起火燃烧，易导致火灾的蔓延，如木柱、木屋架、木梁、木楼梯等。

B3级：易燃性建筑材料，无任何阻燃效果，极易燃烧，火灾危险性很大。
